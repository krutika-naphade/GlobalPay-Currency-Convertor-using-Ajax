# 🌍 GlobalPay – Digital Payment System

## 📌 Project Description
GlobalPay is a web-based digital payment simulation system developed using **Java Servlets and AJAX**. It allows users to perform operations like currency conversion, sending money, QR-based payments, and viewing transaction history.

---

## 🚀 Features
- 🔐 User Login System  
- 💱 Currency Converter (USD, INR, EUR)  
- 💸 Send Money with tax & fee calculation  
- 📱 QR Code Payment  
- 📊 Dashboard with currency trends  
- 📜 Transaction History (tabular format)  
- ⚙️ Profile Settings (edit/save)  

---

## 🛠 Technologies Used
- Java (Servlets)  
- AJAX (Asynchronous communication)  
- HTML, CSS, JavaScript  
- Bootstrap  
- Chart.js  
- QRCode.js
